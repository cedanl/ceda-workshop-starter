# Briefing – CEDA Concept Workshop

## Problem / Opportunity
Maak de eerste conceptversie van de CEDA-website.

## Appetite
30-40 minuten, vast. Geen verlenging.

## Solution
Moet een moderne website zijn. Vrij te kiezen: aantal pagina's en welke elementen je gebruikt.
Implementeer doorlinken naar:
- Github: https://github.com/cedanl
- Npuls: https://npuls.nl/
- Community: https://community-data-ai.npuls.nl/groups/view/44d20066-53a8-48c2-b4e9-be348e05d273/project-center-for-educational-data-analytics-ceda

### Potentiële ideeën
- Een menukaart met wat CEDA te bieden heeft
- https://eduwallet.nl/, een site van ander Npuls project
- https://voorzieningenwijzer.npuls.nl/, een site om aantal verschillende producten te linken aan student en architectuur

## Succescriterium
Een werkend concept dat iemand via git kan pullen en dan werkt op localhost, zonder aanvullende configuratie. 
Bonus (buiten scope, puur als iemand zelf verder wil): op basis van handoff.md + readiness.md een refactor doen en zelf hosten op GitHub Pages.
